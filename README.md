# Screen Blanker

Screen-Blanker is a minimal React-based tool that lets users blank any monitor with a customizable fullscreen color. Ideal for multi-screen setups during movies or focus sessions, it preserves monitor connection while reducing distraction or light spill. Simple UI, quick fullscreen activation, and customizable color options.


# Set-up Commands
git clone https://github.com/riteshbirthal/screen-blanker.git
cd screen-blanker
npm install
npm run dev
